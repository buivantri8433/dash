# Dashboard - Bảng Điều Khiển Đẹp

Một trang dashboard hiện đại, đẹp mắt và responsive được xây dựng bằng HTML, CSS, và JavaScript.

## 🌟 Tính năng

- ✨ Giao diện hiện đại với gradient colors
- 🌙 Dark mode / Light mode toggle
- 📊 Biểu đồ dữ liệu (Chart.js)
- 📱 Responsive design - hoạt động tốt trên mọi thiết bị
- 🎯 Sidebar navigation
- 📈 Thống kê doanh thu, người dùng, lưu lượng
- 📋 Bảng giao dịch gần đây
- 🎨 Màu sắc trang nhã và chuyên nghiệp

## 📁 Cấu trúc tệp

```
project/
├── index.html      # File HTML chính
├── style.css       # Stylesheet
├── script.js       # JavaScript
└── README.md       # File này
```

## 🚀 Cách sử dụng

1. Clone repository
```bash
git clone https://github.com/buivantri8433/dash.git
```

2. Mở file `index.html` trong trình duyệt
```bash
cd dash
open index.html
```

3. Hoặc sử dụng Live Server
```bash
npx http-server .
```

## 🎨 Tùy chỉnh

### Thay đổi màu sắc
Sửa các biến CSS trong file `style.css`:
```css
:root {
    --primary-color: #667eea;
    --secondary-color: #764ba2;
    /* ... */
}
```

### Thêm dữ liệu
Chỉnh sửa dữ liệu trong `script.js`:
```javascript
data: [12000, 15000, 13500, 18000, 16500, 19000, 21000]
```
